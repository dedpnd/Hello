Отсортированный массив

Описание

Страница представляет собой визуализированную сортировку алгоритмом «всплытие». Для реализации использовались JQuery и Animate.css.

При загрузки страница генерирует массив из 30 случайных чисел от 0 до 99, который предстоит отсортировать и вывести его на страницу.

Во время сортировки, самое наименьшее число будет перемещено в регион "Отсортированный массив".
